# CustomGUI
## Type of GUI
- Chest
- Double Chest
- Hopper
- Dispenser
## Commands
- `/gui chest`
- `/gui chest2`
- `/gui doublechest`
- `/gui doublechest2`
- `/gui hopper`
- `/gui hopper2`
- `/gui dispenser`
- `/gui dispenser2`
## How to Setup?
You can edit in plugin_data on plugin folder
