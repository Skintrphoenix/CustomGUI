# CustomGUI
## Have 4 type of gui
- Chest
- Double Chest
- Hopper
- Dispenser
## Commands
- `/gui chest`
- `/gui chest2`
- `/gui doublechest`
- `/gui doublechest2`
- `/gui hopper`
- `/gui hopper2`
- `/gui dispenser`
- `/gui dispenser2`
## How to Setup?
You can edit in plugin_data on plugin folder
## All Dependencies
- [Devirion](https://poggit.pmmp.io/p/Devirion)
- [InvMenu](https://www.mediafire.com/file/wndazlx9h4t6nm4/InvMenu.phar/file)
